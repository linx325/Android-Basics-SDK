# Android项目基础库
## 依赖方式
#### Step 1. 将 JitPack 存储库添加到您的构建文件，将其添加到存储库末尾的根 build.gradle 中：
```
allprojects {
	repositories {
		...
		maven { url 'https://jitpack.io' }
	}
}
```
#### Step 2. 添加依赖
```
dependencies {
	implementation 'com.github.linx325:Android-Basics-SDK:lastVersion'
}
```
